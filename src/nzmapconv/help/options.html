NZMapConv options

The program provides options for changing the format of the converted coordinates and for changing the layout of the coordinate conversion display. These can be accessed by clicking the "Options..." button on the coordinate conversion screeen. If this is not visible (there is an option to hide the buttons), then you can access it by clicking on the icon at the top left of the display (or pressing "Alt+space" and selecting "Options..." from the system menu.

The program options are saved in the windows registry when the program exits, and restored when the program is restarted.

Note: if the options button and menu item are not visible they may have been disabled by a system administrator.

Coordinate format options

These options control the way that converted coordinates are presented. They do not affect input coordinates - they may be entered in any valid format

Map reference coordinates

The is just one option for map reference coordinates, which is whether to use 6 figure or 8 figure references. Six figure references are most commonly used and are accurate to the nearest 100 metres.

Projection coordinates

Projection coordinates have two options. "Order" controls the order coordinates are shown, either easting then northing, or northing then easting. "Decimal places" controls the number of decimal places with which coordinates are displayed. Choosing 0 displays the coordinates to the nearest metre.

Latitude/longitude coordinates

There are three options controlling latitude/longitue coordinates. "Format" chooses whether coordinates are displayed as degrees/minutes/seconds, degrees/minutes, or just decimal degrees. "Order" chooses whether latitudes or longitudes are displayed first. "Decimal places" chooses the number of decimal places to show. This applies to the last part of the coordinate, either seconds, minutes, or degrees according to the selected format.

Layout options

Layout options control the appearance of the program.

Coordinate font

Chooses the font that will be used to display coordinates.

Labels above coordinates

Chooses whether labels appear beside or above the coordinates. Check this box to display labels above the coordinates, or clear it to display labels beside coordinates.

Vertical layout

Chooses whether the coordinates are displayed side by side or as a vertical list.

Coordinate types

Chooses which coordinates will be shown. Only the selected options will be displayed. Options are "Map refs" for displaying map reference coordinates (NZMS260, NZTopo50), "Projection" for displaying projection coordinates (NZMG, NZTM2000), and "Lat/Lon" for displaying latitude/longitude coordinates (NZGD1949, NZGD2000).

Show buttons

Chooses whether the buttons are displayed on the main form. If this is deselected then only the coordinate entry boxes are displayed.

Keep converter on top

Chooses whether the coordinate converter is always visible. If this is selected then it will always be visible, if not then it may be hidden behind other windows.
