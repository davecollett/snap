NZMapConv registry settings

The NZMapConv program stores the user options in under the registry key 

HKEY_CURRENT_USER\Software\LINZ\NZMapConv\Settings

... and if that is not defined, then in ... 

HKEY_LOCAL_MACHINE\Software\LINZ\NZMapConv\Settings

When the program exits it writes back the current users settings.

These settings are fairly obviously related to the settings in the options dialog box.

In addition to the dialog box settings, the registry also stores the location of the NZMapConv window, so that it starts up in the same place as it was last located.

There are two options that can only by set with the registry. These are:

ShowOptionsIf this is 0 then the options dialog is hidden, so that the user cannot change settings.
 
ShowCardOptionIf this is 1 then the options dialog box includes additional options for displaying projection and latitude and longitude in a "CARD" format as used by the 111 call centres. In this format projection coordinates are entered as "EN(2345678,5678901)" and latitude/longitude coordinates are entered as "LL(174.92423,-41.18169)".

Example registry file

[HKEY_CURRENT_USER\Software\LINZ\NZMapConv]

[HKEY_CURRENT_USER\Software\LINZ\NZMapConv\Settings]
"LatLonFormat"=dword:00000000
"LatLonOrder"=dword:00000001
"LatLonDecimalPlaces"=dword:00000002
"ProjectionOrder"=dword:00000001
"ProjectionDecimalPlaces"=dword:00000000
"ProjectionFormat"=dword:00000000
"MapReference8Digigs"=dword:00000000
"StayOnTop"=dword:00000000
"CoordLabelsAbove"=dword:00000001
"VerticalLayout"=dword:00000000
"ShowMapRef"=dword:00000001
"ShowProjCoords"=dword:00000001
"ShowLatLonCoords"=dword:00000001
"ShowButtons"=dword:00000001
"ShowOptions"=dword:00000001
"ShowCardOption"=dword:00000001
