The #gx_enu_error command

#gx_enu_error value value value mm
#gx_enu_error value value value mm correlation corr
#gx_enu_error value value value mm correlation dist1 corr1 dist2 corr2

This specifies the default error for point coordinate observations. The error is specified by a millimetre absolute error (mm) at each point. This can be followed by the correlation between points. The correlation is assumed to be zero between different ordinates (ie East is only correlated with east, north with north and up with up). The correlation can also be specified with by two distances, dist1 < dist2, and two correlations, corr1 > corr2. In this case the correlation is treated as corr1 between points less than dist1 metres apart, corr2 for distances greater than dist2 apart. Between the two ranges the distance is proportional to the inverse of the distance. Snap will only accept positive correlations.
