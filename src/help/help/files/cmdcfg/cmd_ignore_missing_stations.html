The ignore_missing_stations command

ignore_missing_stations [yes|no]

This command instructs SNAP to ignore station codes in the data file which are not listed in the coordinate file. The missing stations will be listed in the error file. If this command is not used then SNAP will report all missing stations in the data files and stop the adjustment.
