The default_refraction_coefficient command

default_refraction_coefficient value

This command defines a value that will be used for refraction coefficients codes that occur in the data file and are not specified by a refraction_coefficient command. The command is equivalent to refraction_coefficient * value

value

Specifies the value that will be used for subsequently defined refraction coefficients. The default value is 0.075.
