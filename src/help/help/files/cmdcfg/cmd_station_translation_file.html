The station_translation_file command

station_translation_file stn_translation_file_name

Defines the name of the station translation file. The station translation defines possible translations of station codes as they are read from command files. Translations can apply for a limited range of dates to allow for a station having been disturbed, so that SNAP can calculate separate coordinates within that range of dates.

Note that all station_translation_file and translate commands are processed in order before any data files are read.

Station translation files can also be specified for individual data files in the data_file command.
